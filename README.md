# 桃李屋



## 简介
[![Static Badge](https://img.shields.io/badge/license-MIT-bule)](./LICENSE)



**桃李屋**  是一个校园论坛，主要用于学生交流，目前前端使用的技术栈为 Vue3和各种第三方包，界面具有基本的动画和美观，后端采用SpringBoot3和SpringCloud兼备分布式处理，还使用了SpringBootWebFlux做socket通信，实现实时聊天室的功能。
